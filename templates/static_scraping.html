Static Web Scraping

Static web scraping is the process of extracting information from websites whose content is directly available in the HTML source. I used Python, Requests and BeautifulSoup to build these projects.

Project Overview

About Static Scraping

Static websites return complete HTML when requested. BeautifulSoup parses the HTML while Requests downloads the webpage. The extracted data is then cleaned using Pandas and stored in CSV files.

Project Workflow

1️⃣ Send Request

Use Requests library to download webpage HTML.

2️⃣ Parse HTML

BeautifulSoup parses HTML into an easy-to-search structure.

3️⃣ Extract Data

Find required elements using tags, classes and CSS selectors.

4️⃣ Export CSV

Store extracted information into CSV files using Pandas.

Features

Fast Extraction

Collects structured information quickly.

Reusable Code

Python scripts can be reused for similar websites.

CSV Export

Stores scraped data in CSV format for future analysis.

Data Cleaning

Pandas helps prepare clean datasets for visualization.

Sample Code

import requests
from bs4 import BeautifulSoup

url = "https://example.com"

response = requests.get(url)

soup = BeautifulSoup(response.text,"html.parser")

title = soup.find("h1").text

print(title)
